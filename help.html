Overview & Features

Michaelsoft Beint is a drawing program that utilizes the HTML canvas element.

Some key features of Michaelsoft Beint include:

  • A dedicated toolbar with 3 tools
  • Separate sizes for the Brush & Eraser
  • Painting with any color you wish
  • Being able to easily clear the canvas
  • Saving your work to your desktop
  • Keyboard functionality with the toolbar

Brush

Keyboard Shortcut: B

The Brush Tool is selected by default when you open Michaelsoft Beint.

The digital brush is like a real life paintbrush, being used to apply color onto the canvas.

Pencil

Keyboard Shortcut: P

The Pencil Tool is like a mini version of the brush.

The Pencil Tool does not have a adjustable size, its thickness is always set to 1.

Eraser

Keyboard Shortcut: E

The Eraser Tool is like a real life eraser, erasing whatever is on the canvas.

The Eraser Tool works by using the same functionality as the brush except for the fact that it is always white, the canvas' color.

Brush Size & Eraser Size

Keyboard Shortcut (Increase Size): ] or +

Keyboard Shortcut (Decrease Size): [ or -

Keyboard Functionality

Every tool in Michaelsoft Beint is paired with a specific keyboard key.

These pairings allow for quick access to the available tools, making your work faster.

The tool-key pairings have been listed for your convenience:

  • Brush | B
  • Pencil | P
  • Eraser | E
  • Brush Size & Eraser Size (Increase) | ] or +
  • Brush Size & Eraser Size (Decrease) | [ or -
  • Clear Canvas | Delete
  • Save Canvas | S
